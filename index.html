Documentos

Gestión interna

Panel administrador para docentes, candidatos, votantes y resultados.

Proyecto institucional

Esta plataforma fue preparada para la I.E. Santa Rosa de Santo Domingo, ubicada en Huarmey, con un enfoque moderno y participativo. La sección de elecciones permite organizar procesos escolares como Municipio Escolar, brigadas u otras consultas internas.

La primera versión funciona como prototipo local. Luego puede conectarse con GitHub Pages y Firebase para que los datos se actualicen en tiempo real.
